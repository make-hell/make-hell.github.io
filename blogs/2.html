How to use Ender3 3D printer

1. This tutorial will gide you on a step-by-step basis on how to use Ender3 3D printer. If you have just received your Ender3, checkout this video below for assembly instruction by CHEP.

2. After Assembly, install a CAD software and think of a design that you want to print. There are multiple CAD designing software that you can use like Solidworks, FreeCAD and AutoCAD.

Tip: There are websites from where you can download models made by someone or even upload your designs at thingiverse.

How to start 3D printing once you have your model ready!

Ender3 or most of the 3D printing machines accept G-Code commands. To do that, you have to slice the STL file for the post-processing. You can select `Save As` option from your CAD software to save an STL file format. There are multiple free software (I use Ultimaker Cura)that does the conversion. Download the software and we can get started on 3D printing your favourate design.

3. Open the Ultimaker Cura and and open/import the model. You should see something similar to the image below. Make sure that the bottom surface is flat. Slice the model and save it on the memory card.

Add On: Please do checkout upgrades for Ender3 (mainly Silent module v4.2.7) and OctiPrint for managing the printing online with the help of a Raspberry Pi.
PLEASE CONSIDER DOWNLOADING "Ultimaker CURA PRINTING PROFILES" WHICH CAN BE EASILY IMPORTED. THIS WILL HELP YOU A LOT IN TIME MANAGEMENT.
