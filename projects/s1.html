Sleep Monitoring IOT System Project was to design a product that takes in the sensor data and send it to a broker in order to receive the data back on server and hence, store data. This project was funded by Dream Cantec Inc. and was collaborated with SFU's School of Mechatronic Systems Engineering. My team designed and worked on this product for 8-months. This project required us to plan how the whole system would work. There were big tasks as in designing a breakout board to test the system and then design the PCB's that would go in the system. at the same time, we were required to work on the user interface using PyQT so the data is easily read-able to the user. The next step was to implement the communication between the device and the server which none of the team members have worked on before. We used the popular MQTT protocol mostly used in IOT devices for future development. The designing of the mechanical system was done after all the internal electronics were ready and was 3D printed. Overall, it was a challange and I'm happy to work with such an amazing team.
