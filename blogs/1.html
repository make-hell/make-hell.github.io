How to use a 3D printer

This tutorial will gide you on a step-by-step basis on how to use a 3D printer.

Install a CAD software and think of a design that you want to print. there are multiple CAD designing software that you can use. I've used solidworks for this tutorial. You can also use FreeCAD which is free to use on multiple platforms.

Tip: There are websites from where you can download models made by someone or even upload your designs like thingiverse.

I have an Ender 3 that I will be using to 3D print the designs. Its specifications are as follows:

  1. Size: 210x210x250 mm
  2. Material: PLA
  3. Cost: CAD $320 (Ender 3) + CAD $37 (PLA Roll)

How to 3D print an STL file

Usually, you have to slice the STL file for the post-processing. This will convert the STL file into a G code that is widely used for CNC machines. There are multiple free software (Ultimaker Cura)that does the conversion. Download the software and we can get started on 3D printing your favourate design.
